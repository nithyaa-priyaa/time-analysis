Program 6 - CMD Encoding & Decoding

Conversion from CMD to RT, Rx/Tx, SA, WC and vice versa

Program Description

This tool allows users to encode and decode 4-digit hexadecimal Command Words (CMD) used in avionics systems.
CMD words follow the format: RT, Rx/Tx, SA, WC.

Manual Example:
Input CMD: FBA3
Binary: F = 1111, B = 1011, A = 1010, 3 = 0011 → 1111101110100011
Extracted:
RT = 11111 = 31
Rx/Tx = 0 → Rx
SA = 11101 = 29
WC = 00011 = 3
